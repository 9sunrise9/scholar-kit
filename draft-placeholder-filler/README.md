# draft-placeholder-filler

识别 Markdown 正文草稿中的【...】占位符，按占位符提示语义完成填充与改写，并将可复用写作规则沉淀到 WritingStyle.md。

## 功能概述

- 扫描并分类占位符（填空型、改写型、写作提示型、标识型）
- 按优先级执行改写：范围指令 > 术语规范 > 信息密度 > 篇幅压缩
- 仅修改占位符对应范围，避免越界改写
- 完成后执行一致性检查（术语统一、命名统一、结构冲突）
- 增量更新 WritingStyle.md（口语到学术替换对、禁用表达、段落规则）

## 适用场景

- 教材、专著、论文草稿中的占位符批量清理
- 文稿中遗留的 TODO 式写作提示统一落地
- 在不改动整体结构前提下做最小化补全与术语规范化

## 触发词示例

- "填充占位符"
- "处理【】"
- "补全草稿"
- "resolve placeholders"
